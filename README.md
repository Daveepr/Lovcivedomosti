# Lovci vědomostí

**Lovci vědomostí** je online vědomostní hra inspirovaná soutěží *Na lovu*.  
Hráč si může vybrat ze tří režimů:

* **Trénink** – neomezené otázky pro zlepšení znalostí
* **Duel s lovcem** – simulace hry proti virtuálnímu soupeři
* **Hráč vs hráč** – lokální multiplayer

Hra je nekomerční, open-source a vytvořená pro zábavu a trénink vědomostí.
